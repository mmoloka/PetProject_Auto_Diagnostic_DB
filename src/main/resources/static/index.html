Добро пожаловать в приложение поиска результатов диагностики автомобилей

Для отображения всех результатов наберите в строке поиска: localhost:8080/repair/ui

Для отображения результатов по *VIN* наберите в строке поиска: localhost:8080/repair/ui/vin/*VIN*

*VIN* - идентификационный номер автомобиля

Для отображения результатов по *VIN* и *GROUP* наберите в строке поиска: localhost:8080/repair/ui/vin/*VIN*/group/*GROUP*

*GROUP* - конструкционная группа в ремонтной документации
